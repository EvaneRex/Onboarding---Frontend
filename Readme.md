# Eksamens projekt

Dette er frontend delen af et eksamensprojekt, på 2 semester af PBA i webudvikling. Projektet strakte sig over 14 dage, og er det sidste store projekt på semestret.

Hvert team bestod af en frontend gruppe og en backend gruppe, som sammen skulle skabe en onboarding prototype i et reallife projekt.

## Dependensies

For at installere alle dependensies, så kør

```bash
npm install
```

**Projektet bruger bla:**

- Vue
- Vite
- Prettier

### Note

Efter at have hentet mappen ned lokalt, kan udviklingsserveren startes med:

```bash
npm run dev
```

## Om projektets opbygning

Projektet er opbygget i Vue og opdelt i adskillige komponenter som hver har en kommentar i toppen der giver et overordnet indblik i hvad den gør og i nogle tilfælde hvilke komponenter den hænger sammen med.
